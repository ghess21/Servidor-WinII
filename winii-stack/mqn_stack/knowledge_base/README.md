# 🎨 Base de Conocimiento - Media Quality Net

## 📁 **Estructura de Carpetas**

### 🏢 **empresa/**
- **perfil/** - Información de la empresa, misión, visión
- **servicios/** - Catálogo de servicios ofrecidos
- **contacto/** - Información de contacto, horarios
- **historial/** - Historia de la empresa, logros

### 🛍️ **productos/**
- **catalogo/** - Catálogo completo de productos
- **precios/** - Lista de precios y tarifas
- **tecnicas/** - Técnicas de impresión disponibles
- **especificaciones/** - Especificaciones técnicas de productos

### 👥 **clientes/**
- **base_datos/** - Base de datos de clientes
- **historial/** - Historial de pedidos y proyectos
- **preferencias/** - Preferencias y gustos de clientes
- **proyectos/** - Proyectos realizados por cliente

### 🎨 **disenos/**
- **plantillas/** - Plantillas de diseño reutilizables
- **inspiracion/** - Ideas y referencias de diseño
- **estilos/** - Guías de estilo y branding

### ⚙️ **procesos/**
- **ventas/** - Proceso de ventas y cotización
- **produccion/** - Proceso de producción
- **calidad/** - Control de calidad
- **entrega/** - Proceso de entrega

### 📋 **reglas/**
- **politicas/** - Políticas de la empresa
- **procedimientos/** - Procedimientos operativos
- **normas/** - Normas de calidad
- **decisiones/** - Criterios de decisión

### 📸 **archivos_fotograficos/**
- **productos/** - Fotos de productos terminados
- **trabajos/** - Fotos de trabajos realizados
- **disenos/** - Fotos de diseños y mockups
- **logos/** - Logotipos y marcas
- **fondos/** - Fondos y texturas
- **clientes/** - Fotos de trabajos para clientes específicos

## 🎯 **Propósito**

Esta base de conocimiento servirá como fuente de información para:
- ✅ **IA del sistema** - Para responder preguntas y tomar decisiones
- ✅ **Workflows de n8n** - Para automatizar procesos
- ✅ **Catálogo automático** - Para generar catálogos dinámicos
- ✅ **Cotizaciones** - Para calcular precios automáticamente
- ✅ **Reportes** - Para generar reportes personalizados

## 📊 **Uso del Sistema**

### **Para el sistema de IA:**
- Los workflows de n8n accederán a esta información
- Se usará para generar respuestas automáticas
- Servirá para tomar decisiones basadas en datos

### **Para el catálogo automático:**
- Las fotos se procesarán automáticamente
- Se generarán descripciones basadas en el contenido
- Se calcularán precios según las especificaciones

### **Para el sistema de ventas:**
- Se usarán las reglas de cotización
- Se accederá al historial de clientes
- Se aplicarán las políticas de descuento

## 🔄 **Mantenimiento**

### **Actualización automática:**
- Los workflows procesarán nuevas fotos automáticamente
- Se actualizará la base de datos con nuevos clientes
- Se registrarán nuevos productos y precios

### **Actualización manual:**
- Documentos de empresa
- Reglas y políticas
- Información de contacto
